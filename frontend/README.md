A community platform where Ethiopians can ask questions and get answers.

## Features
- User authentication with email verification
- Ask and answer questions
- Categories filter
- User profiles with profile pictures
- Edit/delete your own content
- Contact form with email notifications
- Responsive design

## Tech Stack
- **Frontend**: React, React Router, Axios, CSS
- **Backend**: Node.js, Express, MongoDB, Mongoose
- **Authentication**: JWT, Bcrypt
- **Email**: Nodemailer
- **File Upload**: Multer

## Installation
1. Clone repository
2. Install dependencies: `npm install` in both frontend and backend
3. Create `.env` file in backend (see `.env.example`)
4. Run backend: `npm start` in backend folder
5. Run frontend: `npm start` in frontend folder
